# Kurdish Vocab Helper

## Overview

Kurdish Vocab Helper is a small text based C++ program that runs in the terminal.  
It helps the user practice a small set of Kurdish Bahdini words and see a simple map of a few Kurdish cities.  

All data is hard coded and very small. The main goal is to show how to use lists, a queue, a hash table, a tree, and a graph in one program for the CS210 data structures final project.

## Features

Main menu options

1. View all words  
2. Practice words in order  
3. Look up a word  
4. Show category tree  
5. Show city connections  
0. Exit  

### View all words

Prints all vocabulary entries. For each word it shows

* Kurdish word  
* English meaning  
* Category name
